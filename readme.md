## About
Number conversion utilities.

## Usage
```
> dec 10
DEC: 10
HEX: 0A 
BIN: 00001010 
```

```
> hex deadbeef
DEC: 3735928559
HEX: DE AD BE EF
BIN: 11011110 10101101 10111110 11101111
```

```
> bin 1010 0101
DEC: 10
HEX: 0A 
BIN: 00001010 
```
